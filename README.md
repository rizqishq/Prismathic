## How To Start

1. Clone the repository

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npx expo start
   ```

4. Launch the app on your preferred platform:
   - Press `a` for Android
   - Press `i` for iOS
   - Press `w` for web
   - Or just use [Expo Go](https://expo.dev/go)
